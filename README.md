# 📊 Supply Chain Analytics Dashboard using Power BI

## 📌 About This Repository

This repository contains an interactive **Power BI Dashboard** developed as a continuation of my Python-based Supply Chain Analysis project.

In the previous project, extensive **Data Cleaning, Feature Engineering, Exploratory Data Analysis (EDA), Profitability Analysis, Bottleneck Detection, and Root Cause Analysis** were performed using Python.

The insights generated from that analysis were then transformed into an interactive Business Intelligence solution using Power BI to enable KPI monitoring, supply chain performance tracking, and data-driven decision making.

🔗 **Python EDA Repository:**
https://github.com/Toufique-khan/Python-Supply-Chain-Analysis-for-Ecommerce

---

# 🎯 Business Problem

A global e-commerce company manages product fulfillment, shipping, and delivery operations across multiple regions.

The company faces challenges such as:

* Delivery delays
* Inconsistent shipping performance
* Profitability fluctuations
* Regional operational inefficiencies
* Customer dissatisfaction due to late deliveries

The objective of this dashboard is to analyze these challenges and provide actionable insights to improve supply chain performance.

---

# 📂 Dataset Information

The dataset contains information related to:
